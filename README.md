# Word CLI

Command line interface to get useful information from a chunk of text.

## Install

```
npm i -g @simboonlong/word-cli
```

## Count

Count total words.

```
word count --text 'foobar in hello world!'
```

Count total characters.

```
word count --text 'foobar in hello world!' --char
```
